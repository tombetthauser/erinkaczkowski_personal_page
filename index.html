Wacky Projects

    A national survey for visual artists built with D3.js data visualizations and a custom cross-reference visualization tool for the browser.

  • Computer Buddies

    A series of hand-sculpted computer history models cast in concrete, resin and foam. Scanned for 3D printing and made freely available.

    website | github

  • Slacker News

    A Hacker News clone specifically for visual artists to share articles, information, questions etc. Built with Express, PostgreSQL and a simple template-engine (pug) front-end.

    website | github
