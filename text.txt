Replace image files with whatever text you want in the text.txt file. You can add links and also leave lines blank if you want no text. When you add a new image it will default to just show the file name so yo ucan find it easily in the test.txt file. The whole project will rebuild / update after you change anything in the text.txt file. You can still delete images or move them around by changing the record.txt file but you'll have to manually update the text.txt file after or text will probably be displaying in the wrong order.
